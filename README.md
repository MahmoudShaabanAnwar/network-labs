# Network Labs

This repository contains networking labs implemented using Cisco Packet Tracer.

## Labs Included
- Static Routing
- Default Routing
- RIP v2
- OSPF Single Area
- DHCP on Router
- DHCP Snooping
- VLAN Basic Configuration
- Etherchannel
- Portsecurity
- router on stick
- SVI on multiple layer switch
- Dynamic NAT

## Tools
- Cisco Packet Tracer

## What I Learned
- Configuring static and dynamic routing protocols
- Implementing DHCP services and security
- Applying basic network security concepts
- Troubleshooting using Cisco CLI
- Configuring VLANs and assigning access ports
- Implementing trunk links between switches
- Bundling multiple physical links using EtherChannel
- Securing switch ports with Port Security
- Inter-VLAN routing using Router-on-a-Stick
- Configuring SVI on a multilayer switch for inter-VLAN routing
- Implementing Dynamic NAT to translate private IPs to public IPs
