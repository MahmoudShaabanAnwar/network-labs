# Network Labs

This repository contains networking labs implemented using Cisco Packet Tracer.

## Labs Included
- Static Routing
- Default Routing
- RIP v2
- OSPF Single Area
- DHCP on Router
- DHCP Snooping

## Tools
- Cisco Packet Tracer

## What I Learned
- Configuring static and dynamic routing protocols
- Implementing DHCP services and security
- Applying basic network security concepts
- Troubleshooting using Cisco CLI
